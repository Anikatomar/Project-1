# Book Recommendation Chatbot
The Book Recommendation Chatbot is an intelligent virtual
assistant designed using IBM Watson Assistant to help users
discover books based on their preferred genre or author. The
chatbot provides an interactive and user-friendly experience,
making book exploration fun and personalized.

It enable natural conversation through AI-powered dialogue
and encourage continuous engagement by suggesting similar
or related books.
It support for multiple user queries in a single conversation
(context handling and digression).

TOOLS USED:

1. IBM CLOUD-IBM Watson Assistant for creating intents, entities, and dialog flows
2. VS CODE-Compiling html and css codes for website building
3. CHAT GPT-To get list of books of various genres and authors.
4. MICROSOFT EXCEL-CSV files to store training examples for different intents (e.g., greeting, thank you, book recommendation)
